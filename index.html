Kelinci Framework

Framework UI modern berbasis SASS dan jQuery.

1. Grid System

Gunakan class .kelinci-row dan .kelinci-col untuk layout grid.

Kolom 1
Kolom 2
<div class="kelinci-row">
    <div class="kelinci-col">Kolom 1</div>
    <div class="kelinci-col">Kolom 2</div>
</div>

2. JavaScript Kelinci

Contoh script jQuery untuk modal.

$(document).ready(function() {
    $(".kelinci-open-modal").on("click", function() {
        let target = $(this).data("target");
        $(target).addClass("show");
    });

    $(".kelinci-close-modal").on("click", function() {
        $(this).closest(".kelinci-modal").removeClass("show");
    });
});
